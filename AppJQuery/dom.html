DOM (Document Object Model)

O DOM, sigla de (Document Object Model) é uma interface que representa como os documentos HTML e XML são lidos pelo seu browser. Após o browser ler seu documento HTML, ele cria um objeto que faz uma representação estruturada do seu documento e define meios de como essa estrutura pode ser acessada. Nós podemos acessar e manipular o DOM com JavaScript, é a forma mais fácil e usada.
A estrutura do jQuery Mobile tem um mecanismo simples para manter o DOM organizado: sempre que uma página é carregada via Ajax, ela é sinalizada para remoção do DOM quando você navega para outra página (tecnicamente, em pagehide). Se você retornar a uma página excluída, o navegador poderá recuperar o arquivo de seu cache ou solicitará novamente o servidor, se necessário. No caso de listas aninhadas, removemos todas as páginas que compõem a lista aninhada quando você navega para uma página que não faz parte da lista. As páginas incluídas em uma configuração de várias páginas não serão afetadas por esse recurso - somente as páginas trazidas pelo Ajax são gerenciadas dessa maneira pelo jQuery Mobile.
