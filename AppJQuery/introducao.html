App Jquery Mobile

O que é o JQuery Mobile

jQuery Mobile é um framework de interface gráfica (UI) em plataforma cruzada para smartphones e tablets, construído sobre as fundações da biblioteca jQuery. [...] O JQuery Mobile utiliza uma estrutura de CDNs espalhados pelo mundo todo para garantir uma conectividade dos recursos disponibilizados em diversos apps. Outra vantagem que o JQuery Mobile oferece aos desenvolvedores de app híbridos é que as atualizações de estilos e comportamentos são realizadas em segundo plano, sem a necessidade da criação de novas versões dos aplicativos
