O uso de Grids

Os grids são elementos não visuais do Jquery Mobile que permitem dividir a área de trabalho em vários espaços.
A vantagem de se trabalhar com grids e que cada um dos espaços é calculado de forma responsiva e qualquer o tamanho dos objetos dentro de cada uma das células do grid são proporcionais a ele.
