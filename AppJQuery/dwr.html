Design de Web Responsivo

O Jquery Mobile é uma estrutura de interface de usuario baseada em Jquery que funciona em todas as plataformas populares de telefones, tablets, e-readers e desktop.
Construído com acessibilidade de e acesso universal em mente, seguimos os princípios de aprimoramento progressivo e Responsive Web Design (RWD).
A configuração baseada em marcação de HTML5 facilita o aprendizado, mas uma API poderosa facilida a personalização profunda da biblioteca
Dois importantes recursos do HTML ganham uma grande maior relevancia na responsividade de imagens na versão 5, os parametros WIDTH e HEIGTH e o conceito de contenção em grades (GRID).
